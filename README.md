# Next2D AI SKILL

## 日本語
Next2D Framework および Next2D Player を用いた開発を強力にサポートするAI Skillsです。
TypeScriptベースの型安全な実装、WebGLレンダリングを最大化するパフォーマンス最適化、そして直感的なアニメーション実装のためのコード生成を行います。

- TypeScript First: Next2D特有のクラス構造やAPIに完全対応。
- Smart Debugging: 描画エラーや論理エラーの迅速な特定と解決。
- Performance Tips: WebGL/Canvasの負荷を抑えた実装提案。

## English
AI Skills that powerfully support development using the Next2D Framework and Next2D Player.
It provides TypeScript-based type-safe implementation, performance optimization to maximize WebGL rendering, and code generation for intuitive animation implementation.

- TypeScript First: Fully compatible with Next2D's unique class structure and APIs.
- Smart Debugging: Rapidly identify and resolve rendering errors and logical errors.
- Performance Tips: Suggest implementations that minimize WebGL/Canvas load.
